# 企业数据资产管理系统 - 快速原型

## 🚀 原型概述

### 核心假设
**主要假设**：企业需要一个统一的平台来管理、发现、治理和保护数据资产，帮助数据团队提升数据质量和协作效率。
**成功指标**：
- 用户能在3分钟内找到目标数据资产
- 核心用户流程完成率超过80%
- 用户反馈收集率达到30%以上

### 最小可行功能
**核心流程**：用户登录 → 数据地图浏览 → 资产详情查看 → 收藏关注

**功能集**（按优先级排序）：
1. 用户认证与个人工作台
2. 数据地图与资产搜索
3. 资产详情与元数据展示
4. 数据血缘可视化
5. 资产收藏与订阅
6. 用户反馈收集
7. 基础分析仪表盘

**技术栈**：Next.js 14 + TypeScript + Tailwind CSS + shadcn/ui + Supabase

## 🗄️ 数据库设计

### 核心数据模型

```sql
-- 用户表
CREATE TABLE users (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    email TEXT UNIQUE NOT NULL,
    name TEXT,
    avatar_url TEXT,
    role TEXT DEFAULT 'viewer', -- admin, editor, viewer
    department TEXT,
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

-- 数据源表
CREATE TABLE data_sources (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name TEXT NOT NULL,
    type TEXT NOT NULL, -- mysql, postgresql, hive, etc.
    connection_config JSONB,
    owner_id UUID REFERENCES users(id),
    created_at TIMESTAMP DEFAULT NOW()
);

-- 数据资产表
CREATE TABLE data_assets (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    guid TEXT UNIQUE NOT NULL,
    name TEXT NOT NULL,
    description TEXT,
    type TEXT NOT NULL, -- table, file, api, metric
    source_id UUID REFERENCES data_sources(id),
    schema_definition JSONB,
    security_level TEXT DEFAULT 'internal', -- public, internal, sensitive, confidential
    quality_score DECIMAL(5,2) DEFAULT 0,
    usage_count INTEGER DEFAULT 0,
    owner_id UUID REFERENCES users(id),
    tags TEXT[],
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);

-- 资产收藏表
CREATE TABLE asset_favorites (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id),
    asset_id UUID REFERENCES data_assets(id),
    created_at TIMESTAMP DEFAULT NOW(),
    UNIQUE(user_id, asset_id)
);

-- 血缘关系表
CREATE TABLE lineage_edges (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    source_asset_id UUID REFERENCES data_assets(id),
    target_asset_id UUID REFERENCES data_assets(id),
    transformation_type TEXT, -- ETL, SQL, API
    transformation_detail TEXT,
    created_at TIMESTAMP DEFAULT NOW()
);

-- 资产访问记录表
CREATE TABLE asset_access_logs (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id),
    asset_id UUID REFERENCES data_assets(id),
    action TEXT, -- view, search, favorite, export
    created_at TIMESTAMP DEFAULT NOW()
);

-- 反馈表
CREATE TABLE feedback (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id),
    content TEXT NOT NULL,
    rating INTEGER CHECK (rating >= 1 AND rating <= 5),
    category TEXT, -- bug, feature, general
    status TEXT DEFAULT 'pending', -- pending, reviewed, resolved
    created_at TIMESTAMP DEFAULT NOW()
);

-- 通知表
CREATE TABLE notifications (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(id),
    title TEXT NOT NULL,
    content TEXT,
    type TEXT, -- info, warning, success
    is_read BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP DEFAULT NOW()
);
```

## 🎨 页面结构设计

### 页面列表
1. **首页/工作台** - 个人仪表盘，展示关键指标、最近访问、收藏资产
2. **数据地图** - 资产目录树和卡片视图

4. **资产详情** - 完整资产信息、字段、血缘图
