### 5. 验证码发送功能实现 (P0)

**文件**: 
- `AuthService.java` - 添加接口
- `AuthServiceImpl.java` - 实现发送逻辑
- `AuthController.java` - 调用实现

**修复前**:
```java
// TODO: 实现验证码发送逻辑
return Result.success();
```

**修复后**:
```java
@Override
public void sendResetCode(String account) {
    // 检查发送频率限制
    checkRateLimit(account);
    
    // 查找用户
    User user = findUserByAccount(account);
    
    // 生成6位验证码
    String code = generateVerificationCode();
    
    // 保存到Redis
    saveCodeToRedis(account, code);
    
    // 发送验证码（邮件/短信）
    sendCodeByEmail(user.getEmail(), code);
    // 或
    sendCodeBySms(user.getPhone(), code);
}
```

**功能**:
- 验证码频率限制（60秒间隔）
- 验证码有效期（10分钟）
- 邮箱/手机号自动检测
- 账户信息掩码保护
