### 核心改进：Feign 调用链路贯通

edams-asset 现在作为资产聚合核心，通过 Feign 客户端与下游服务深度整合：
- **createAsset** → 自动触发 lineageFeignClient.createNode() 注册血缘节点
- **createAsset** → 自动触发 workflowFeignClient.startProcess() 发起审批流
- **createAsset** → 自动发布 Kafka 事件至 dam.asset.created
- **getAsset** → 聚合 qualityFeignClient.getAssetQualityScore() 质量评分
- **getAsset** → 聚合 lineageFeignClient.getLineageGraph() 血缘图谱
