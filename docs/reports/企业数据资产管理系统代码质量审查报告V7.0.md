### 1. Flowable工作流引擎集成 (P0)

**文件**: `WorkflowServiceImpl.java`

**修复前**:
```java
// TODO: 实际应调用Flowable RepositoryService部署BPMN XML

```

**修复后**:
```java
@Autowired
private RepositoryService repositoryService;

@Override
public WorkflowDefinition deploy(Long definitionId, String operator) {
    // 使用Flowable RepositoryService部署BPMN XML
    Deployment deployment = repositoryService.createDeployment()
        .name(def.getName())
        .key(def.getCode())
        .addString(def.getCode() + ".bpmn20.xml", def.getBpmnContent())
        .deploy();

    def.setProcessDefKey(deployment.getKey());
    def.setProcessDefId(deployment.getId());
    def.setStatus(1);
    def.setPublishedTime(java.time.LocalDateTime.now());
    definitionMapper.updateById(def);
}
```

**影响**: 工作流发布现在真正部署到Flowable引擎，支持完整的工作流执行

### 4. 通知收件人信息自动填充 (P0)

**文件**: `NotificationServiceImpl.java`

**修复前**:
```java
// TODO: 根据receiverId查询用户邮箱/手机号等地址信息
notif.setTargetAddress("");  // 空地址导致发送失败
```

**修复后**:
```java
@Autowired
private UserFeignClient userFeignClient;

private void enrichNotificationTargetAddress(Notification notification) {
    var userInfo = userFeignClient.getUserById(notification.getReceiverId());
    if (userInfo == null) return;

    String channel = notification.getChannel();
    if ("email".equalsIgnoreCase(channel)) {
        notification.setTargetAddress(userInfo.getEmail());
    } else if ("sms".equalsIgnoreCase(channel)) {
        notification.setTargetAddress(userInfo.getPhone());
    }
}
```

**影响**: 通知发送时自动填充收件人联系方式

@FeignClient(name = "edams-user", path = "/api/v1/users")
public interface UserFeignClient {
    @GetMapping("/{id}")
    Result<UserBasicInfo> getUserById(@PathVariable("id") Long id);

    @Data
    class UserBasicInfo {
        private Long id;
        private String username;
        private String realName;
        private String email;
        private String phone;
        private String departmentName;
    }
}
